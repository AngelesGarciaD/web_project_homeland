# Nombre del proyecto:

Tripleten web_project_homeland

# Descripción:

Este proyecto es un sitio web en dónde se muestran las ciudades de donde son originarios algunos colaboradores de TripleTen.

# Tecnologías usadas:

- HTML
- CSS Grid
- Flexbox
- Metodología BEM
- Media Queries para el diseño responsivo.
